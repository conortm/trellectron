# trellectron

An [Electron](https://electron.atom.io/) wrapper for [Trello](https://trello.com/).

Download the [latest release](https://github.com/conortm/trellectron/releases/latest).

## Running locally

```bash
git clone git@github.com:conortm/trellectron.git
cd trellectron
yarn
yarn run start
```
